How to do this, a complete guideÂ¶

Starting steps:Â¶

  1. Initialize a mice object

mice = Mice((256, 4), full_load=True, load_allen=True)

  1. Fit parcellation

mice.parcel.fit_parcel(force_fit=True, cutoff=2)

  1. Transform masks from CCF to individual spaces

from register.register import Register
reg = Register(mice)
reg.register_anat(save_results=True)

Next steps:Â¶

  1. Copy masks to Yale

  1. Run ./do_xform.sh on cuda5

  2. Copy resulting 2d masks to _copy directory

  1. Copy masks from _copy directory to the masks_dir

  1. Run this to fix holes in 2D object masks

  1. Run organize_func (you can do this from terminal, it calls mice.cat_data() as well)
