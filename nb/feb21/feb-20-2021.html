Generate networks¶

These are the networks i build:

  • ER: the famous Erdos-Renyi network, aka binomial network

  • M: a modular network. it has some closely connected modules with dense adjacency structure. These cores are sparsely connected to other modules. There is no real distance between the core modules, every core is equally conected to other cores.

  • HM: Hierarchichal modular network. Just like M, but the modules are organized in a hierarchical way. I will plot the adjacency matrices below and things will become more clear

  • C: configuration model based on M. This is a network that has identical degree sequence with M, but it is randomized

See plots below.

# HIDE CODE

num_blocks = 10

a_hm = get_adjacency(block_size=block_size, num_blocks=num_blocks, mode='HM', random_state=RANDOM_STATE)

g_m = nx.from_numpy_matrix(a_m)
g_hm = nx.from_numpy_matrix(a_hm)

# get configuration model

# get ER graph
g_er = erdos_renyi_graph(n=g_c.order(), p=density(g_c))

Community structure¶

Time for fun stuff. In this section I will study behavior of different community detection algorithms on simulated networks. First, let’s start with a classic, but not so great algorithm. In this section we will use another network measure known as modularity:

  • Modularity: \(Q = \frac{1}{N}\sum_i\)

Girvan-Newmann algorithm¶

This was the first time someone publishd on community detection. According Newmann this algorithm is not great, but their paper gets cited due to introducing a new question to ask: given a graph or network, can we partition it into colsely-knit communities?

# HIDE CODE


# run algorithm
communities_gn = {}
max_k = 60
for k, g in tqdm(graphs.items(), leave=False):
    comp = nx_comm.centrality.girvan_newman(g)
    limited = itertools.takewhile(lambda c: len(c) <= max_k, comp)
    comm_dict = {}
    for idx, comm in enumerate(limited):
        comm_dict[idx] = tuple(sorted(c) for c in comm)

    communities_gn[k] = comm_dict


# process results
modularities_gn = {}

        modul_dict[i] = nx_comm.modularity(g, communities=comm_list)
        comm_dict[i] = len(comm_list)

    num_communities_gn[k] = comm_dict

Louvain algorithm¶

Wants to directly maximize Q, modularity

# HIDE CODE


# run algorithm
communities_louvain = {}
max_k = 1000
for k, g in tqdm(graphs.items(), leave=False):
    tmp = {}
    for resolution in tqdm(np.logspace(-2, 1, max_k+1), leave=False):
        partition = community_louvain.best_partition(g, resolution=resolution, random_state=RANDOM_STATE)
        
        comm = collections.defaultdict(list)
        for node, v in sorted(partition.items()):
            comm[v].append(node)  
        tmp[resolution] = dict(comm)
        
    communities_louvain[k] = tmp


# process results
modularities_louvain = {}
num_communities_louvain = {}

for k, g in graphs.items():
    modul = {}
    num_comm = {}
    for resolution, partition in communities_louvain[k].items():
        community_list = list(partition.values())
        modul[resolution] = nx_comm.modularity(g, communities=community_list)
        num_comm[resolution] = len(community_list)
    
    modularities_louvain[k] = modul
    num_communities_louvain[k] = num_comm

        community_dict = list(communities_louvain[name].values())[idx]
        community_list = list(community_dict.values())
        palette = sns.color_palette('tab20', n_colors=len(community_list))
        num_incontiguous = 0
        for ll, (comm_list, color) in enumerate(zip(community_list, palette)):
            contiguous_list = []
            for k, g in itertools.groupby(enumerate(comm_list), lambda ix: ix[0]-ix[1]):
                contiguous_list.append(list(map(itemgetter(1), g)))

            if len(contiguous_list) > 1:
                num_incontiguous += 1

            for item in contiguous_list:
                rect = patches.FancyBboxPatch(
                    (item[0],) * 2,
                    item[-1] - item[0],
                    item[-1] - item[0],
                    linewidth=4,
                    edgecolor=color,
                    facecolor='none',
                    boxstyle="round,pad=1",
                    ls='--' if ll%2==0 else '-',
                )
                axes[j, i].add_patch(rect)

        msg = "{:s}, res = {:.2f}   /   Q = {:.3f}\n# communities:  {:d}  (# incontiguous: {:d})"
        msg = msg.format(
            name, list(modularities_louvain[name].keys())[idx],
            list(modularities_louvain[name].values())[idx],
            len(community_list), num_incontiguous,
        )
        axes[j, i].set_title(msg, fontsize=13, y=1.02)
